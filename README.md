**AudioStudio** — многодорожечный редактор звуковых файлов для создания, редактирования и обработки аудиоматериала. Визуальный интерфейс для работы со звуковыми дорожками и набор встроенных инструментов обработки.



## Архитектура системы

Проект состоит из трёх компонентов:

**AudioStudio** — ядро приложения: визуализация амплитудных графиков, управление воспроизведением и временной шкалой, многодорожечная структура, пользовательский интерфейс, взаимодействие с файловой системой.

**AudioBridge** — нативная библиотека: низкоуровневая обработка звука в реальном времени, применение звуковых эффектов через цепочку обработки, высокая производительность при работе с потоками аудиоданных.

**AudioStudioInstaller** — инструмент для распространения готового продукта, формирует пакет установки для конечного пользователя.



## Возможности

### Работа с файлами
Поддержка форматов **WAV, MP3, FLAC, OGG, M4A, AIFF, WMA, AAC** · Встроенный проводник с обзором дисков и папок · Перетаскивание файлов на дорожки · Отображение длительности и размера файлов

### Многодорожечная структура
Неограниченное количество дорожек · Независимое управление каждой дорожкой · Визуальное отображение амплитудных графиков · Индикатор текущей позиции воспроизведения

### Редактирование
Вырезание, копирование, вставка, удаление · Система отмены и возврата действий · Выделение фрагментов мышью · Применение изменений к выделенным областям

### Воспроизведение
Воспроизведение, пауза, остановка · Перемотка к началу и концу · Отображение текущего времени и общей длительности · Синхронизация визуального индикатора

### Инструменты обработки
Фильтр низких частот · Фильтр высоких частот · Усиление · Эхо · Реверберация

### Масштабирование
Изменение масштаба временной шкалы · Изменение масштаба интерфейса · **Ctrl + колёсико мыши** для быстрого изменения масштаба области дорожек



## Технические подробности

**Библиотеки:** NAudio, WPF, WiX Toolset, Windows API через AudioBridge

**Требования:** Windows 10+, .NET 8.0+, x64

**Хранение данных:** Временные данные дорожек в оперативной памяти · Кэширование амплитудных графиков · Буфер обмена для фрагментов
